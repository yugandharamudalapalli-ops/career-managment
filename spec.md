# Career Information Management System

## Overview
A web-based system for managing student career information with both administrative and public access capabilities.

## Core Features

### Admin Dashboard
- Secure admin interface for managing all system data
- Overview of total students and career paths
- Quick access to add new records

### Student Profile Management
- Add new student profiles with personal information
- Update existing student records
- Delete student profiles
- Store academic details (degree, GPA, graduation year)
- Store career information (career path, skills, experience level)
- Store contact information

### Search and Filter System
- Search students by name
- Filter students by career path
- Filter students by qualifications/degree
- Filter students by graduation year
- Combined filtering options

### Public Career Information View
- Browse available career paths
- View student profiles associated with each career path
- Read-only access to student information
- No administrative functions in public view

## Data Storage (Backend)
The backend must store:
- Student profiles with personal, academic, and career details
- Career path categories and descriptions
- Admin authentication credentials

## Backend Operations
- CRUD operations for student profiles
- Search and filtering logic
- Data validation for student information
- Admin authentication verification

## User Interface
- Clean, professional design suitable for educational/career management
- Responsive layout for desktop and mobile access
- Clear navigation between admin and public sections
- Form-based interfaces for data entry
- Table/grid views for displaying student lists
- Search and filter controls
